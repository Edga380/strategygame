# Strategy game
Earth needs more resources to sustain mankind. After finding a planet with a lot of it they think their problems are solved. While building a new base on the planet and harvesting its resources they encounter a species that have been harvesting it too, so the fight for resources begins.
#
## Why I chose to develop this game?
A little bit of history. When I was a little kid I went to a friends place he had 'Sega genesis 2' and a game called 'Dune' I got hooked on it and ended up playing that game for hours that day.
Later on in my life I discovered that there is other versions of that game, the one I played the most and still sometimes play is called 'Emperor battle for dune'.
So after developing 'Bubble space wars' I started thinking about developing similar game to 'Dune' with unity but never had time to do it.
So when I had to develop a game for FAC I straigt away though about this game and that I really want to code it even though it's going to be really hard.
